# MyLauncher v1.2.0

A new update is available! This version focuses on improving project information, blocked apps control, and performance optimization.

## 🚀 What's New

### ℹ️ Project Information
We've added a new "About" dialog and an info button. Now you can easily access project links, developer social media, and contact options directly from the app.

### ⏳ Improved Blocking Control
Stay focused! It is no longer possible to permanently remove blocks. You can now only adjust the "Time's Up" duration, preventing you from forgetting to re-block the application and helping you maintain your digital well-being.

### ⚡ Performance Optimizations
Technical adjustments have been made to the build configuration to improve overall performance and app efficiency.

## 🛠️ Technical Details
*   **Version:** `1.2.0` (Version Code: `3`)
*   **Minimum SDK:** Android 12 (API 31)
*   **Target SDK:** Android 15 (API 36)
*   **Build Tools:** Compiled with SDK 36 (minor 1).
*   **Technologies:** Built entirely with **Jetpack Compose** and Kotlin.
