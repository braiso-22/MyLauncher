# 🛠️ Development Guide

Welcome! If you're a developer and want to build **MyLauncher** yourself or contribute to the project, follow these steps.

## 📋 Prerequisites

- **Android Studio Panda 2** (or newer).
- **Android Min SDK 31**.
- **JDK 17**.

## 🚀 Building the App

1. **Clone the repository:**
   ```bash
   git clone https://github.com/braiso22/MyLauncher.git
   ```
2. **Open the project:**
   Launch Android Studio and select **Open**, then navigate to the cloned folder.
3. **Sync Gradle:**
   Wait for Android Studio to finish indexing and syncing the Gradle files.
4. **Run:**
   Select a device or emulator and press the **Run** button (green play icon).

## 🏗️ Tech Stack

- **UI:** Jetpack Compose.
- **Data:** Room Database & DataStore.
- **Language:** Kotlin.
- **Pattern:** MVVM.
